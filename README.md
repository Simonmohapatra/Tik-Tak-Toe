# Tik-Tac-Toe (Frontend)
A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

##  Features

* Two-player gameplay (X vs O)
* Winner detection
* Draw detection
* Reset game functionality
* New game functionality
* Responsive user interface
* Clean and beginner-friendly code structure

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎮 How to Play

1. Open the game in a browser.
2. Player O starts first.
3. Players take turns placing their marks on the board.
4. The first player to align three marks horizontally, vertically, or diagonally wins.
5. If all cells are filled and no player wins, the game ends in a draw.
